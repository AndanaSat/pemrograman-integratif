# Pemrograman-Integratif-A

## Steps

### 1. Clone github repo in local
  ```git clone https://github.com/AndanaSat/Pemrograman-Integratif-A.git```

### 2. Install library
  ```npm i grpc```
  
  ```npm i @grpc/proto-loader```
  
  ```npm i mysql2```
  
### 3. Run
  Run server.js
  
  ```npm start```
  
  Run run.js
  
  ```node run.js```
